Morgana is a metalanguage for language designers: easily define the syntax of your language and its abstract syntax tree, at the same time. 

morg Morgana 

start Morg 

node Morg := (
  name: [ Keyword::Morg Identifier ], 
  start_symbol: [ StartStatement ],
  statements: { Statements }
) 

...
Based on the simple idea of lifting wsn-like rules to the type level, allows you to generate an entire front-end from a single source of truth. Simply write a morg and generate data structures, parsers and much more in your preferred programming language.
